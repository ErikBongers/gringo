Gringo extension opties

Globale instellingen
Deze instellingen hebben invloed op alle gebruikers en foute instellingen kunnen de werking van Gringo verstoren.

Plaats hier de tags die beschikbaar zijn voor alle gebruikers. Elke tag op een nieuwe regel. Het formaat per regel is 4 comma-gescheiden waarden: 

tagnaam, omschrijving, tekstkleur, achtergrondkleur
...waarbij de kleuren geldige CSS color values moeten zijn. De tagnaam is verplicht, de andere waarden zijn optioneel.
